Sécurité & Confidentialité

Transparence totale sur le traitement de vos emails et la sécurisation de vos identifiants.

1. Vos Identifiants & Accès

Outlook / Microsoft

Méthode : OAuth2 (Moderne)

Nous n'avons jamais accès à votre mot de passe. Nous utilisons un "jeton d'accès" sécurisé fourni par Microsoft. Vous pouvez révoquer cet accès à tout moment depuis votre compte Microsoft.

Gmail

Méthode : Mot de passe d'application

Vous générez une clé unique de 16 caractères dédiée uniquement à Mial. Cette clé est stockée de manière chiffrée (AES-256) dans notre base de données sécurisée.

Autre (IMAP)

Méthode : Standard Chiffré SSL/TLS

La connexion se fait via le protocole standard IMAP sécurisé. Vos identifiants sont chiffrés en base de données avec une clé unique d'encryption par utilisateur.

2. Traitement des Données (IA)

Nous utilisons l'intelligence artificielle pour générer vos résumés, en suivant une politique stricte de confidentialité.

Flux Sécurisé

  • Récupération : Mial récupère les emails via connexion chiffrée (HTTPS).
  • Anonymisation : Les données sont pré-traitées sur nos serveurs.
  • Traitement OpenAI : Seul le contenu nécessaire est envoyé à l'API OpenAI via une connexion Entreprise.

Zero Data Retention

Garantie Commerciale : Les données envoyées à l'API d'OpenAI ne sont pas utilisées pour entraîner leurs modèles publics (ChatGPT). Elles sont supprimées après traitement.

3. Infrastructure & Confidentialité

Serveurs VPS Sécurisés

Hébergement en Europe sur des serveurs privés isolés (VPS).

Base de Données Isolée

Accès restreint. Backups chiffrés quotidiens.

Pas de Revente

Nous ne vendons, ne louons et ne partageons aucune donnée client à des tiers publicitaires.
